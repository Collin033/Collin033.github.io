## [A - 最大的公约数](https://vjudge.net/contest/526444#problem/A)

当n为偶数时，其最小的因子只能是2

当n为奇数时，其最小的因子只能是3

那么假设n为偶数，取a = n，b = n/2，则n/2即为答案

假设n为奇数，那么n - 1为偶数，所以（n - 1）/2为答案

（因为最小的因子只可能是3，n/3 明显小于等于 (n - 1)/2）

## [G - 木棍](https://vjudge.net/contest/526444#problem/G)

大意：给定n个木棍的长度length和重量weight，第一根木棍的加工长度为1分钟。而后对于每根木棍stick[i],如果下一根木棍的长度和重量均大于该stick[i]，则不需要花费额外的时间加工，否则还是需要1分钟。

1. 对木棍的长度(或重量)进行升序排序，同时保证长度相等的情况下，重量也是升序排序的。
2. 定义一个标记数组flag[]，用于记录该木棍是否被加工。
3. 从头开始取木棍，使用两个变量l,w来记录当前加工的木棍的长度和重量。加工完一个木棍后，遍历后序的木棍(注意是否已经被加工)，如果符合length[j] >= l && weight[j] >= w 则更新l,w，并将该木棍标记为已加工。

主体代码：

```text
int ans = 0;
for(i = 1 to n){
    ans++;
    for(j = i + 1 to n){
         if(length[j] >= l && weight[j] >= w){
             flag[j] = 1;
              l = length[j],w = weight[j];
        }
    }
}
```
